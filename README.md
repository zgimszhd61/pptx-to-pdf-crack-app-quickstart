# pptx-to-pdf-crack-app-quickstart
## 问题：PPTX文件如何转换成PDF文件
 - 使用方法
```
import aspose.slides as slides


# 加载PPTX文件
pres = slides.Presentation("example.pptx")

# 将PPTX转换为PDF
pres.save("example.pdf", slides.export.SaveFormat.PDF)

print("PPTX文件已成功转换为PDF!")
```
